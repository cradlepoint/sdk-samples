# system_monitor

![Python](https://img.shields.io/badge/Python-3.8-yellow)

Gets various system diagnostics, alerts on thresholds, and puts current status in the asset_id field.

## What It Monitors

- CPU usage (alerts at 80%)
- Memory usage (alerts at 80%)
- Uptime
- Ethernet port status including LAN (alerts on port status change)

Blocks of code show examples for monitoring parameters so you can write your own.
